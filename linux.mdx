## Preinstalled on

UbuntuCE - a Linux-based free and open source operating system geared towards Christians.
Ubuntu Studio - a free and open source operating system made for creative people.

# snapcraft

sudo snap install presenterbuddy

# .AppImage

1. Allow executing: Right-click the .AppImage file, and choose "Properties"
2. Properties: Go to "Permissions", and check the box that says "Allow executing file as program"
3. Execute: Double-click the .AppImage file

# arm64 devices

The default download does not support "arm64" devices. But you can download a custom build from the downloads page, labeled "X-arm64.AppImage".
