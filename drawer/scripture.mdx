Choose from over 200 bible versions from API.Bible, or import you own local file from Zefania, OSIS, Beblia, or OpenSong in XML format. Search the web for available scriptures you can use (e.g. "XML Bible Download KJV"), make sure you have the proper rights to use them. In the center you can find all of the books, chapters and verses.

# Search
Start searching (CTRL/CMD + F) to quickly select any verse in the bible, it will auto complete the book name. Can be like this: "John 1:1-4", or like this: "Genesis 1:1-2+5", then press Enter to show in output.

Also in the bottom there is a search icon you can use to search for content in the entire bible.

# Selecting
You can select verses manually by holding CTRL/CMD while clicking on them. Double clicking any verse will show it in the output. Pressing Enter when verses are selected will output them directly. Holding CTRL/CMD and pressing Arrow left/right will move the selection to the next verses.

# Create a collection
Select two or more of the scriptures in the left drawer panel, right click and press "New collection". Collections are found in the same panel. Selecting one will preview only the first one in the center, but selecting it you can see that it's showing multiple versions at the same time.

# Options
On the right you have many options to change how the scriptures are displayed. Here you can also change the template used. Please note that for collections the templates must have at least as many textboxes as there are scriptures.

# Create a show
On the right side you can see a preview of what the output will look like. Here you can tweak some settings, and change the template. Click "New show" to create a normal show from the scripture.

The "Max verses per slide" means the maximum amount of bible verses per slide before creating a new one. Say this is set to 3, and you select ten verses. Then you will get four slides, three with three verses and the last with one verse.

# FAQ
Q: How can I customize the scripture layout?

A: To customize the scripture, all you need to do is edit the "Scripture" template in the "Templates" drawer tab. You can also create a new different template and select that on the right side in the "Scriptures" drawer tab.

Q: The importing is not working

A: Make sure you have selected the correct option, you can try multiple if you are unsure. If none of them work, please try this converter which supports many more formats, or take contact.

Q: The book names are in English, how to change?

A: PresenterBuddy takes the Bible book names from the XML file. But sometimes they are in English for a different language or missing entirely, in that case you have to change the book names manually and import the file again: Edit the XML file, and add a name="" attributes to the book tags, like this: `<book name="Genesis".`

Q: How to use Bibles offline

A: To use Bibles that does not rely on an internet connection or API.Bible. You need to import your own XML Bibles. To do that click "New scripture" in the bottom left of the "Scriptures" drawer, and find a Bible version available for use on the web of one of the supported file types.
