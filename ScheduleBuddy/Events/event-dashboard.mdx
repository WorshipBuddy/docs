# Event Dashboard

The Event Dashboard provides an operational view for managing your event during setup and execution. It gives you quick access to all event management features in one place.

## Accessing the Event Dashboard

To access the event dashboard:

1. Go to the **Services** page
2. Click on your event
3. The dashboard provides tabs for different aspects of event management

Alternatively, you can access the dashboard directly via the event dashboard URL (if you have it bookmarked).

## Dashboard Overview

The dashboard shows:

### Event Header
- **Event Name**
- **Dates** - Start and end dates
- **Location**
- **Status Badge** - Draft, Published, Active, Completed, or Archived

### Quick Stats
Statistics about your event:
- Total registrations
- Registration trends
- Other key metrics

### Status Workflow
Visual indicators showing the progress of event setup:
- Event details completed
- Registration form configured
- Services added
- Other setup milestones

### Quick Actions
Buttons to quickly access:
- Manage Services
- View Registrations
- Manage Logistics (if enabled)
- Event Settings

## Dashboard Tabs

The dashboard organizes features into tabs:

### Registration Tab
- View all registration responses
- Export registration data
- Manage registrations

### Services Tab
- View and manage event services
- Schedule teams for services
- Share service schedules

### Transportation Tab (if enabled)
- Manage transportation assistance
- Track flights
- Assign drivers and routes
- View transportation dashboard

### Housing Tab (if enabled)
- Manage housing assignments
- Track accommodation needs
- Assign guests to hosts
- View housing dashboard

### Meals Tab (if enabled)
- Plan meal periods
- Track meal attendance
- Manage dietary requirements
- View meals dashboard

### Check-In Tab
- Manage check-in stations
- Track arrivals and departures
- View check-in logs
- QR code scanning (if enabled)

<Info>
**Note**: Tabs appear based on what features you've enabled in your event settings. If you haven't enabled housing, transportation, or meals, those tabs won't appear.
</Info>

## Event Settings

From the dashboard, you can access event settings:

1. Click **"Settings"** button in the header
2. Or use the **"Event Settings"** quick action
3. Modify event details, registration settings, and service configurations

<Info>
**Note**: Only Owners and Organization Admins can access event settings.
</Info>

## Event Status Workflow

The dashboard shows the event's current status in the workflow:

1. **Draft** - Event is still being set up
2. **Published** - Event is live and registration is available
3. **Active** - Event is currently happening
4. **Completed** - Event has finished
5. **Archived** - Event is archived for historical reference

The status can be changed in the event settings at any time.

<Info>
**Tip**: Use the status indicators to see what still needs to be done for your event setup.
</Info>

## Managing During the Event

During the event, the dashboard helps you:

- **Track registrations** - See who's registered and their information
- **Manage logistics** - Coordinate transportation, housing, and meals
- **Handle check-ins** - Track arrivals and departures
- **View services** - See service schedules and assignments
- **Monitor status** - Keep track of event progress

## Best Practices

1. **Use the dashboard** - It's your central hub for event management
2. **Check regularly** - Review registrations and logistics regularly as the event approaches
3. **Update status** - Keep event status current (draft → published → archived)
4. **Export data** - Regularly export registration data for backup and analysis
5. **Coordinate logistics** - Use the logistics tabs to manage transportation, housing, and meals effectively
