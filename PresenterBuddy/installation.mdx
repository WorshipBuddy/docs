---

PresenterBuddy is available for download and installation on macOS and Windows operating systems.

## Installation on macOS

1. Download the [.dmg file](#)
2. **Install**: Double click the `.dmg` file
3. **Copy**: Drag the "PresenterBuddy" icon into the "Applications" folder
4. **Open**: Find and open the app

## Installation on Windows

1. Download the [.exe file](#)
2. **Install**: Double click and run the `.exe` file after it's been downloaded to your PC
3. **Follow Prompts**: Follow the on-screen prompts
4. **Run Anyway**: Click "Run Anyway" if Microsoft Defender SmartScreen issues an unrecognized app warning
5. **Open**: Find and open the app
