PresenterBuddy is everything you need to display lyrics, Bible verses, and announcements on a second screen. It's clean, fast, and easy to use with zero-to-minimal training for your church volunteer team.

## List of Features

Here are some of the main features of PresenterBuddy:

- Instant lyric search with in-song highlighting to pinpoint exactly where it appears
- Instant Bible verse search with in-verse highlighting to pinpoint exactly where it appears
- Add sets created in WorshipBuddy directly into your presentation queue
- Auto-detect multi-lingual versions of any song for instant display or addition to queue (when Bilingual Version is turned on)
- Multiple song databases depending on region
- Instant switching between 7 Bible versions and counting
- Easy connection to your church's official songbook with Songbook ID / WorshipBuddy integration (no more manual uploading of new church songs)
- Advanced stanza selection in Presentation Queue to allow for the right part of the song to be displayed when switching
- Blue Bar selection/resizing option allows for displaying select lines within or in-between a stanza in the song or verse
- Song Import Tool with chord cleanup for minimal editing of new songs outside of church songbook
- Line selection available for lyrics and Bible verse
- Announcements Tab for Creating and Presenting Church Announcements
- Cross compatibility with Melodie Database and other SQLite databases for easy import into PresenterBuddy
- Option to present "Song Info Overlay" for a set duration on any position of the output screen
- Adjustable Margins
- Font selection, font sizing and alignment available for lyrics, Bible verse and Bible verse reference
- Line spacing adjustable for lyrics and Bible verse
- Option for blank line insertion between stanzas for lyrics or Bible verse
- Option to hide stanza tag lines (e.g. "Chorus")
- Adjustable fade duration for lyrics and Bible verse
- Install recall of entire Bible chapter to the Presentation Controller section for easy transition to other verses within the chapter
- Automatically song numbering of newly imported or added songs (no more guessing of available song numbers)
- Presentation History with Date and Time stamps
- Drag to reorder Presentation Queue
- Smooth transitions from song to song with Previous and Next Buttons in the Presentation Queue
- Gray shading on stanzas to denote what's NOT being displayed
- Auto-Detection of Presentation screen with option to manually switch output screens
- Over the Air (OTA) Updates forever (until the Rapture of the church)

## System Requirements

| Requirement | Details |
|---|---|
| OS | macOS 10.15+ (Catalina, 2019) / Windows 10+ (64-bit) / Linux x64 |
| CPU | x64 (Intel/AMD 64-bit) |
| RAM | ~4 GB minimum, 8 GB recommended |
| Storage | ~400–500 MB installed |
| Display | Any resolution; secondary display recommended for projection |
| Network | Required for song sync with WorshipBuddy; not needed for offline use |
