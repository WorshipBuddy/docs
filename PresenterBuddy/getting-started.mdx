## SongBook ID

After installation, open up PresenterBuddy. When you open the application for the first time, you will be prompted to enter a SongBook ID. This allows you to link directly to your church's official Songbook. Enter your SongBook ID to access your church organization's songbook (format is `ABC123`).

You can choose to skip this step and go back to it later in the settings. If this step is skipped, you will get this warning:

<Warning>No Songbook ID connected. Showing public domain songs only. Add your SongbookID in Settings to access your church's full library.</Warning>

Open up Settings using the "Gear" icon in the top right-hand corner of the application and go to the **General** tab. Under **Songbook**, enter your Songbook ID and click **"Connect"**.

## Default General Settings

Open up Settings using the "Gear" icon in the top right-hand corner of the application and go to the **General** tab. In the **Default** section of the General tab, you can choose your default song database and default Bible version. This is what will be loaded by default any time you use PresenterBuddy.

By default, the Bilingual Version is set to "None". You can change this to the language that you want to switch to during the live service. For example, if the Bilingual version is set to Spanish and the Song Database default is English, PresenterBuddy will automatically find equivalent songs (if available) in Spanish when an English song is being displayed.

## Database Management

Open up Settings using the "Gear" icon in the top right-hand corner of the application and go to the **General** tab. In the **Database** section of the General tab, you can choose to Update, Reset, Import or Export your database.

### Update Database

Click on **"Update Database"**. When connected to the internet, this will pull in the latest and official Church Songbook databases. This only needs to be done once or whenever your church organization has updated their Songbook.

### Reset Database

Click on **"Reset Database"** to delete all locally added songs and overwrite everything with fresh data from the server. Only do this once you have exported the PresenterBuddy database.

<Warning>There is no undo button once the database has been reset.</Warning>

### Import Database

Click on **"Import Database"** to import your PresenterBuddy database (which you previously exported). The PresenterBuddy database has a `.pbdata` file extension. Find the desired database and click **"Open"**.

If your organization previously used an SQLite database (e.g. Melodie Presentation Software uses a `.s3db` file extension) you can import that into PresenterBuddy by switching the dropdown in the Import Tool to include **"All Files"** or **"Melodie/SQLite Database"**. Find the desired database and click **"Open"**.

### Export Database

Click on **"Export Database"** to export your PresenterBuddy database to your local computer.

## Adjust Display Settings

Open up Settings using the "Gear" icon in the top right-hand corner of the application and go to the **Appearance** or **Text** tabs to modify your display settings. In the **Appearance** section, you can adjust your display screens, backgrounds and presentation specifics. In the **Text** section, you can adjust fonts, sizing, alignments and spacing for Songs and Bible verses.
