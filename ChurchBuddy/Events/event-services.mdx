# Services in Events

Events can include services (worship services, sessions, meetings, etc.) managed within the event context — separate from regular Calendar services.

## Understanding Event Services

When you create a service within an event:

- It is stored as part of the event
- It appears in **Event Dashboard → Services**
- It does **not** appear in **Calendar → Services**
- Teams are scheduled via bulk scheduling filtered to this event

## Adding a Service to an Event

1. Open the **Event Dashboard** for your event
2. Click the **Services** tab
3. Click **Add Service**
4. Fill in:
   - **Service Name** (e.g., "Opening Session", "Worship Service")
   - **Start Date and Time**
   - **End Date and Time**
   - **Location** (optional)
   - **Teams** - Select involved teams
5. Configure recurrence if needed (same rules as regular services, max 20 occurrences)
6. Save

## Managing Event Services

### Editing

1. **Event Dashboard → Services**
2. Click **Edit** on the service card (**Owner / Org Admin only**)
3. Save changes

### Removing

1. **Event Dashboard → Services**
2. Delete the service
3. Confirm removal

<Warning>
Removing a service also removes all team assignments for that service.
</Warning>

## Scheduling for Event Services

1. Open **Event Dashboard → Services**
2. Click **Schedule Teams** on any service card (**Owner / Org Admin only**)
3. The **Schedule Team Members** page opens, filtered to this event's services only
4. Scroll through service cards and assign people to positions (auto-saves)
5. Return to the Event Dashboard when finished

<Info>
This uses the same bulk scheduler as regular services (`/org-schedule-team/`), but filtered to the current event.
</Info>

## Sharing Service Schedules

Share who's serving at each event service:

1. **Event Dashboard → Services**
2. Click **Share Schedule** (**Owner / Org Admin only**; modal title: "Share Service Team Schedules")
3. Copy and share the public link

## Best Practices

1. **Name clearly** - e.g., "Friday Night Worship" or "Session 1: Welcome"
2. **Set times accurately** - Include start and end times
3. **Assign teams early** - Schedule well in advance
4. **Use consistent locations** - Set at event level if all services share a venue
