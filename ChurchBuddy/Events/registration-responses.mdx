# Registration Responses

View, manage, and export registration responses from the Event Dashboard.

## Viewing Responses

1. Open the **Event Dashboard** for your event
2. Click the **Registration** tab
3. Open the **Registrations** section

You'll see a list with **Submitted**, **Registrants**, **Services**, and **Actions** by default. Use **Columns** to show **Name** and **Email**.

## Response Information

For each registration:

- **Registrant Information** — Name, email, contact details
- **Responses** — Answers to all form questions
- **Registration Date** — When they registered

Registrants receive an edit link in their confirmation email when **Allow editing responses** is enabled.

## Editing Responses

**Registrants:** Use the edit link in their confirmation email to update answers or remove individual party members (when multiple registrations are allowed).

**Owners / Organization Admins:** Click **Edit** in the Actions column — opens `/event-response/`.

Users with granular `registration_edit` can edit the **Form Builder** but do not get the per-row **Edit** button on the registrations list.

See also the [My Event portal](/ChurchBuddy/Events/my-event).

## Exporting Responses

1. Open **Event Dashboard → Registration → Registrations**
2. Click **Export CSV**
3. The file downloads with each question as a column and each registration as a row

<Info>
Use the **Columns** toggle to show or hide table columns. There is no search/filter bar in the registrations list today.
</Info>

## Anonymous Submissions

If anonymous submissions are enabled, registrants may hide their identity from **other registrants** — organizers still see name and email in the dashboard list and CSV export.

## Registration Statistics

The **Analytics** tab shows registration trends and summary statistics. See [Event Analytics](/ChurchBuddy/Events/analytics).

## Best Practices

1. **Export regularly** — Back up registration data
2. **Review responses** — Check for incomplete or incorrect information
3. **Communicate with registrants** — Use exported emails for event updates
4. **Respect privacy** — Handle personal information appropriately
