# Equipment Resources

The **Resources** sub-tab under **Calendar** lets you manage equipment and inventory — separate from **Rooms** (physical spaces). Track items, check them out, and schedule maintenance.

<Info>
**Platform**: Web only. Resources are at **Calendar → Resources**, not in the mobile app.
</Info>

## Overview

Equipment Resources provides:

- **Resource inventory** - Catalog equipment, supplies, and assets
- **Checkout/check-in** - Track who has what
- **Maintenance tracking** - Flag items needing service
- **Categories and status** - Organize and filter your inventory
- **Room assignment** - Link resources to specific rooms

## Accessing Resources

1. Click **Calendar** in the top navigation
2. Click the **Resources** sub-tab
3. Available to Owners and Organization Admins

See [Rooms](/ChurchBuddy/Setup/rooms) for physical space booking — that is a separate feature.

## Adding a Resource

1. Click **Add Resource**
2. Enter details:
   - **Name** - Item identifier
   - **Category** - Grouping (AV, Furniture, etc.)
   - **Description** (optional)
   - **Serial number / asset tag** (optional)
   - **Status** - Available, Checked Out, Assigned, Maintenance, Out of Service, etc.
3. Save

## Resource Status

| Status | Meaning |
|--------|---------|
| **Available** | Ready to check out |
| **Checked Out** | Currently assigned to someone |
| **Assigned** | Permanently or semi-permanently assigned to a room (detail view shows "Assigned to Room") |
| **Maintenance** | Being serviced (badge label; not "In Maintenance") |
| **Out of Service** | Not usable |

## Checking Out Resources

1. Find the resource in the grid or list view
2. Click to open details
3. Check out to a person with optional due date and notes
4. The resource status updates to **Checked Out**

## Checking In Resources

1. Open **Checked Out** from the action bar to see active checkouts
2. Select the checkout to return
3. Check in the resource — status returns to **Available**

## Maintenance

1. Click **Maintenance Due** in the action bar
2. View items flagged for maintenance
3. Update status when maintenance is complete

## Views and Filters

- **Grid View** / **List View** toggle
- **Category filter** - Filter by resource category
- **Status filter** - Filter by availability status

## Permissions

| Role | View | Check Out/In | Manage Resources |
|------|------|--------------|------------------|
| Owner | ✓ | ✓ | ✓ |
| Org Admin | ✓ | ✓ | ✓ |
| Others | ✗ | ✗ | ✗ |

## Best Practices

1. **Use categories** - Group similar items for easier filtering
2. **Track serial numbers** - Helps identify specific units
3. **Set maintenance schedules** - Keep equipment in good condition
4. **Review checkouts regularly** - Follow up on overdue items
