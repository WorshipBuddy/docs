# BibleBuddy Integration

ChurchBuddy integrates with BibleBuddy to allow you to create reading plans for your organization. Members can then join these plans in the BibleBuddy app to track their Bible reading progress.

## What is BibleBuddy Integration?

The BibleBuddy integration allows you to:

- Create reading plans for your organization
- Set up reading schedules with timelines and frequency
- Create custom plans with specific books and chapters
- Track organization-wide progress on reading plans
- Control who can create and manage plans

<Info>
**Note**: Reading plans created here appear in the BibleBuddy app, where organization members can join them and track their personal reading progress.
</Info>

## Access Requirements

To access BibleBuddy integration settings, you must be:
- An **Owner** of the organization, OR
- An **Organization Admin**, OR
- A user who has been granted access through BibleBuddy Access Control

## Setting Up Access Control

Owners and Admins can control who has access to create and manage reading plans.

### Access Control Options

1. **Owner/Admin Access** - Owners and Organization Admins always have access
2. **Team-Based Access** - Allow specific teams to access BibleBuddy
3. **Individual Access** - Allow specific users to access BibleBuddy

### Granting Team Access

1. Navigate to `/org-biblebuddy/?org={your_org_id}`
2. In **Access Control**, select teams
3. Set the org-wide **View Only** / **Edit** radio (one level for all grants)
4. Click **Save Access Control**

### Granting Individual Access

1. Open the BibleBuddy page
2. Select individual users in **Access Control**
3. Use the same org-wide access level radio
4. Click **Save Access Control**

<Info>
**View Only** / **Edit** is stored org-wide. **+ Create Plan** is hidden from non-admins. **+ Create Custom Plan** is visible to all users with page access and can be saved by any of them today. **Delete** buttons appear for all users with page access. **Edit Plan** shows "coming soon" for everyone — not yet implemented.
</Info>

## Creating Reading Plans

You can create two types of reading plans:

### Standard Reading Plan

A standard plan automatically schedules reading through the entire Bible based on timeline and frequency settings.

1. Click **"+ Create Plan"** button
2. Enter a **Plan Name** (e.g., "2025 Bible Reading Plan")
3. Set the **Start Date** - Choose "Today" or "Tomorrow" or select a specific date
4. Choose **Timeline** mode:
   - **Duration** - Set how long the plan runs (e.g., 365 days, 1 year)
   - **End Date** - Set a specific end date
5. Set **Reading Frequency**:
   - How many times per day/week/month members will read
   - For example: "1 Per Day" means one reading session per day
6. Click **"Create Plan"**

The plan will automatically schedule all Bible chapters across the timeline you specified.

### Custom Reading Plan

A custom plan lets you choose specific books and chapters to include.

1. Click **"+ Create Custom Plan"** button
2. **Step 1**: Enter a **Plan Name**
3. **Step 2**: Set **Start Date** and **End Date**
4. **Step 3**: **Select Reading Content**:
   - Filter by Testament (All, Old Testament, New Testament)
   - Check the boxes next to books you want to include
   - You can select entire books or specific chapters
5. **Step 4**: Set **Reading Frequency** (times per day/week/month)
6. Click through the steps and click **"Create Plan"** when finished

<Info>
**Tip**: Custom plans are great for focused studies on specific books or portions of Scripture.
</Info>

## Viewing Plans

All created plans appear in the "Organization Reading Plans" section. Each plan card shows:

- **Plan Name**
- **Duration** or timeline setting and **reading frequency**
- **Starts:** start date
- **Organization Progress** — overall completion bar

## Plan Actions

For each plan, you can:

- **Plan Details** — See the full reading schedule, dates, and chapters
- **Org Progress** — See individual member progress and overall completion
- **Edit Plan** - Modify plan settings (coming soon)
- **Delete Plan** - Remove the plan (cannot be undone)

<Warning>
**Warning**: Deleting a plan removes it for all organization members. Members will no longer be able to access it in the BibleBuddy app.
</Warning>

## Tracking Progress

### Organization Progress

The organization progress bar shows:
- Total chapters completed across all members
- Overall completion percentage
- Number of active users participating

### Individual Member Progress

To see individual progress:

1. Click **Org Progress** on a plan
2. You'll see a list of organization members
3. For each member, you can see:
   - How many chapters they've completed
   - Their completion percentage
   - Their progress bar
4. Members who haven't joined BibleBuddy will show "Not signed into BibleBuddy" with an invite option

<Info>
**Note**: Progress is tracked when members use the BibleBuddy app. They must join the plan in the app and mark chapters as complete for progress to appear here.
</Info>

## Best Practices

1. **Plan ahead** - Create plans well in advance so members can join before the start date
2. **Use clear names** - Name plans descriptively (e.g., "2025 Year-Long Plan", "Advent Reading Plan")
3. **Set realistic frequencies** - Consider how much reading members can realistically complete
4. **Share plans early** - Let members know about new plans so they can join in the BibleBuddy app
5. **Monitor progress** - Check organization progress periodically to encourage participation

## Troubleshooting

**Can't find BibleBuddy:**
- BibleBuddy is **not in the main org top navigation** on most pages (the nav link is commented out), but the **BibleBuddy page itself** shows a BibleBuddy nav button when you have access
- Navigate directly to `/org-biblebuddy/?org={your_org_id}` while logged in
- Check that you're an Owner, Admin, or have been granted individual/team access

**Plans not appearing in BibleBuddy app:**
- Make sure the plan has been created successfully
- Members need to be part of your organization in BibleBuddy to see organization plans
- Plans may take a few moments to sync to the BibleBuddy app

**Can't create or edit plans:**
- **+ Create Plan** is Owner/Org Admin only (button hidden for others)
- **+ Create Custom Plan** is available to all users with page access
- **Edit Plan** is marked "coming soon" for all roles — not yet implemented

**Progress not updating:**
- Members must use the BibleBuddy app to track their reading
- Progress updates when members mark chapters as complete in the app
- There may be a slight delay before progress appears here
