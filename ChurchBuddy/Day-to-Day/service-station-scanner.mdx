# Service Station Scanner

The Service Station Scanner provides a dedicated check-in interface for services. Set up tablets or computers at check-in stations to scan QR codes or manually look up attendees.

<Info>
**Platform**: Web URL only. There is no dedicated station-scanner screen in the mobile app — open the URL in a mobile browser on a tablet if needed.
</Info>

## Overview

Service Station Scanner is designed for:

- **Dedicated stations** - Pre-configured check-in points with a station ID
- **QR code scanning** - Fast check-in by scanning member QR codes
- **Manual lookup** - Search for attendees by name when needed
- **Multi-station support** - Run multiple stations for busy services

## Setting Up a Station

Stations are created under **People → Check-in** in your organization. Each station gets a unique **station ID**.

### Station URL

Open the scanner with your organization ID and station ID:

```
/service-station-scanner/?org=YOUR_ORG_ID&station=STATION_ID
```

Both parameters are **required**. There is no global nav link — bookmark this URL on each check-in device.

<Info>
This page is now part of the unified scanner. The link above still works and redirects to `/scan/?mode=service-station` (preserving your `org` and `station` parameters). You can also open it directly at `/scan/?mode=service-station&org=YOUR_ORG_ID&station=STATION_ID`.
</Info>

### First-Time Flow

1. Open the station URL above
2. Select **Today's Service** from the dropdown (services for today load automatically)
3. Tap **Continue**
4. Choose **QR Scanner** or **Search by Name**, then tap **Continue**

<Info>
The URL uses `station`, not `service`. The service is selected on the first screen after the page loads.
</Info>

## Check-in Methods

### QR Code Scanning

1. Select **QR Scanner** and tap **Continue**
2. Allow camera access when prompted
3. Point the camera at the member's QR code
4. The system automatically checks them in
5. Confirmation appears on screen

Members display their QR code from the ChurchBuddy mobile app (**Dashboard → Settings → tap your avatar**).

### Search by Name

1. Select **Search by Name** and tap **Continue**
2. Type the person's name or email in the search box
3. Select the correct person from results
4. They are checked in immediately

## Station Interface

### Check-in Confirmation

When someone checks in, the screen shows:

- **Person's name** - Confirms who was checked in
- **Success indicator** - Green checkmark
- **Ready for next** - Returns to scan mode automatically

### Already Checked In

If someone tries to check in twice:

- Shows they're already checked in
- Displays when they checked in
- No duplicate entries created

## Multi-Station Setup

For large services, set up multiple stations:

1. Create each station under **People → Check-in**
2. Give each device its own `station=STATION_ID` URL
3. All check-ins sync to the selected service record
4. Station ID is stored with each check-in

## Best Practices

1. **Test before service** - Verify cameras and connectivity
2. **Train volunteers** - Make sure station operators know both methods
3. **Position stations well** - Place at natural entry points
4. **Have backup plan** - Know how to do manual lookup if scanning fails
5. **Use tablets** - Tablets work well as dedicated stations

## Troubleshooting

**"Missing required parameters":**
- URL must include both `org` and `station`

**Camera not working:**
- Check browser permissions for camera access
- Try refreshing the page

**QR code not scanning:**
- Improve lighting or use manual lookup

**No services in dropdown:**
- Ensure there are services scheduled for today in the organization
