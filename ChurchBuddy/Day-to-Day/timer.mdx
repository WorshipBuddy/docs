# Service Timer

The Timer feature lets you create countdown timers for services with a rundown of segments. Share the display URL with your production team or display it on screens.

## Overview

The Timer provides:

- **Rundown Management** - Create a list of timed segments for your service
- **Countdown Display** - Full-screen countdown view for display on screens
- **Color Indicators** - Visual warnings as time runs low
- **Shareable Display** - Separate display URL for stage monitors or production

## Creating a Timer

1. Go to **Timer** from your organization
2. Name your timer (e.g., "Sunday Service", "Youth Night")
3. Add segments to your rundown
4. Share the display link with your team

## Building a Rundown

A rundown is a list of segments that make up your service. Each segment has:

- **Name** - What this segment is (e.g., "Worship", "Announcements", "Message")
- **Duration** - How long the segment should last
- **Notes** - Optional notes for the production team

### Adding Segments

1. Click **Add Segment** or the **+** button
2. Enter the segment name
3. Set the duration (minutes and seconds)
4. Add any notes (optional)
5. Repeat for all segments

### Reordering Segments

Drag and drop segments to reorder them in your rundown.

### Editing Segments

Click on any segment to edit its name, duration, or notes.

## Running the Timer

### Starting the Timer

1. Click the first segment to select it
2. Click **Start** or press the play button
3. The timer counts down from the segment duration

### Timer Controls

- **Start/Pause** - Start or pause the current countdown
- **Reset** - Reset the current segment to its full duration
- **Skip** - Move to the next segment
- **Previous** - Go back to the previous segment

### Color Indicators

The timer display changes color based on remaining time:

| Color | Meaning |
|-------|---------|
| **Green** | On time - plenty of time remaining |
| **Yellow** | Warning - time is running low |
| **Red** | Overtime - segment time has expired |

## Display View

The display view shows a clean, full-screen countdown for stage monitors or displays.

### Getting the Display URL

1. Click **Open Display** or **Share Display**
2. Copy the display URL
3. Open on another device or browser window

### Display Features

- Full-screen countdown
- Current segment name
- Next segment preview
- Color-coded time indicators
- Auto-advances to next segment

<Info>
**Tip**: Open the display view on a separate monitor or iPad for your stage display, while controlling the timer from your main screen.
</Info>

## Timer Settings

### Auto-Advance

When enabled, the timer automatically moves to the next segment when the current one ends.

### Warning Times

Configure when the timer changes to yellow (warning) and red (overtime):

- **Warning at** - Seconds remaining to show yellow (default: 60 seconds)
- **Critical at** - Seconds remaining to show red (default: 0 seconds/overtime)

## Use Cases

### Service Planning

1. Create your service rundown in advance
2. Set realistic durations for each segment
3. Share with worship leaders and speakers
4. Use during service to stay on schedule

### Production Team

1. Display the timer on stage monitors
2. Production team controls from the booth
3. Speakers and worship leaders see remaining time
4. Keep services running on schedule

### Rehearsals

1. Run through your service with timed segments
2. Identify segments that need more or less time
3. Adjust durations based on actual run times

## Best Practices

1. **Add buffer time** - Include transition segments between main items
2. **Test your display** - Check the display view before service
3. **Communicate** - Make sure all participants know the timer is running
4. **Be flexible** - The timer is a guide, not a strict rule
5. **Save rundowns** - Reuse rundowns for recurring service formats
