# Crowd Meter

The Crowd Meter provides a simple way to count attendance during services using large tap-friendly buttons. Perfect for greeters, ushers, or anyone tracking how many people enter a service.

## Overview

Crowd Meter is designed for:

- **Real-time counting** - Tap to add or subtract from the count
- **Mobile-friendly** - Large buttons work great on phones and tablets
- **Service-specific** - Track counts per service
- **Multiple counters** - Different people can count at different entrances

## Using Crowd Meter

### Setup

1. Go to **Crowd Meter** from your event or service
2. Select the **Event** you're counting for
3. Select the **Service** within that event
4. Start counting

### Counting

The interface shows two large buttons:

- **Green (+)** - Tap to add one person
- **Red (-)** - Tap to subtract one person (for corrections)

The current count displays prominently in the center.

### How It Works

1. Position yourself at an entrance
2. Tap **+** each time someone enters
3. If you miscount, tap **-** to correct
4. The count syncs to the service record

## Multi-Counter Setup

For large venues with multiple entrances:

1. Have a counter at each entrance
2. Each counter uses Crowd Meter on their device
3. All counts for the same service aggregate together
4. View the total on the event dashboard

<Info>
**Note**: Multiple counters adding to the same service will have their counts combined automatically.
</Info>

## Viewing Results

After the service, view attendance totals on:

- **Event Dashboard** - See counts for all services
- **Service Detail** - See the specific service count
- **Reports** - Include in attendance reports

## Best Practices

1. **Brief your counters** - Make sure everyone knows which service to select
2. **Test first** - Do a quick test tap before service starts
3. **Use stable internet** - Ensure devices have good connectivity
4. **Assign entrances** - One counter per entrance prevents double-counting
5. **Count consistently** - Decide whether to count only adults, or everyone including children

## Tips

- The buttons are designed for quick, repeated tapping
- The count persists even if you navigate away
- Use the **-** button for corrections, not to track people leaving
- For children's ministry, use separate check-in tracking
