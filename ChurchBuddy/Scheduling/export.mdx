# Exporting Schedules

ChurchBuddy allows you to export schedules to Word (.docx) or Excel (.xlsx) for sharing, printing, or backup.

## Accessing Export

1. Click **Calendar** in the top navigation
2. Click the **Calendar** sub-tab (not Services)
3. Click **Export Schedule**
4. The **Export Schedule Calendar** popup appears

<Info>
The Export button is only visible on the **Calendar** sub-tab, and only for users with scheduling permissions (Owner, Org Admin, Team Admin, or Scheduler).
</Info>

## Selecting Services to Export

In the export popup:

1. A preview lists **upcoming** services only (past services are excluded)
2. Check each service to include (no Select All/Clear buttons)

## Export Formats

### Export to Word (.docx)

- Each team on its own page
- Dates as column headers; position names as rows; assignee names in cells

### Export to Excel (.xlsx)

- Each team as a separate sheet/tab
- Same layout: dates as columns, positions as rows, assignee names in cells

<Info>
PDF export is not currently available in the UI.
</Info>

## Export Permissions

- Owners
- Organization Admins
- Team Admins
- Schedulers

<Info>
The export includes **all organization teams** in the file — it is not filtered to only the teams you manage.
</Info>

## Best Practices

1. **Export regularly** - Create periodic backups
2. **Use Word for printing** - Use Excel for data analysis
3. **Name files clearly** - e.g., `Schedule_January_2026.docx`
4. **Review before sharing** - Check the file looks correct

<Warning>
Exported schedules contain names and personal information. Handle according to your organization's privacy policy.
</Warning>
